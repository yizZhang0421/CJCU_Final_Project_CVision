/login POST
form:['mail','name'] (google display name)
return:OK or error code

/model_list POST
form:['mail']
return:OK or errorcode or json(id,name,acc,loss)

/create_model POST
form:['mail','name'] (model name)
return:OK or errorcode

/delete_model POST
form:['mail','id'] (model id)
return:OK or errorcode

/model_label_list POST
form:['id'] (model id)
return OK or errorcode or json(id,name) (label id and label name)

/create_label POST
form:['mail','name','id'] (label name and model id)
return OK or errorcode

/delete_label POST
form:['mail','id','label_id'] (model id and labe id)
return OK or errorcode

/label_images POST
form:['mail','id'] (label id)
return OK or errorcode or json(model_id,add_label,image) (add_labe就是label id)
*這裡的image非base64字串，只是檔名，等於只有得到server檔案系統的路徑
是為了要讓圖片載入非同步，才不會影響使用者體驗

/get_image() POST GET
parameters:mail,model,label,image
return jpg bytes
等於瀏覽器打http://[ip:port]/get_image?mail=[mail]&model=[model]&label=[label]&image=[image]
就會顯示圖片
所以可以每格同時加載，幾萬張資料才可以快速顯示
